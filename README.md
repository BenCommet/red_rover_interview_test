## Getting Started
1. Clone the repo
2. Open the directory in a terminal
3. Run `npm install`
4. Run `npx expo start`
5. Type `w` to run the react native web app version of the application. You may need to give permission in order for your terminal to be able to open your browser
6. (Optional) Scan the QR code using a phone with the Expo Go app installed to test the mobile version. If you haven't used Expo before, this will take some setup, but it is not required to test the requirements of the project.
